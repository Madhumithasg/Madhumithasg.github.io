Projects

Smart Billing System For Fruits And Vegetables

Faculty Guide: Asst.Prof T.kalaiselvi , Department of ECE, KEC.

This proposed system, the edible item is not taken to any pre-process .Whereas, a real time image of edible item is captured by the camera placed on the billing desk. The captured image is compared to the database based on the color and texture of the fruits and vegetables. After comparison, the fruit/vegetable kept for billing is identified and a computerized bill is generated with the list of fruit/vegetable with respective cost and weight. Thereby generating a smart bill without human intervention

Technologies and Languages used: Image Processing and Python

Automatic Weed Remover

Faculty Guide: Asst.Prof.S.Sathesh , Department of ECE, KEC.

The main objective of the project is to reduce the difficulty in removing the weeds in the field. This project develops the automated weed remover by using the single board computer (SBC) with open source tools. By using the controller we are controlling the motors to run the machine and finding the weeds using real-time image processing and remove it by using mechanical setup.

Language used: Image Processing and Python
